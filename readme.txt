Integration between Konduto and Vindi

== Description ==

Utilize o metodo antifraude da Konduto integrado com a plataforma VINDI.

O plugin foi desenvolvido de maneira independente. E note que este plugin foi feito baseado na documentação da api disponivel em http://docs.konduto.com/pt/

= Compatibilidade =

Compatível com a versão 2.6.x e 3.x.x do WooCommerce.
